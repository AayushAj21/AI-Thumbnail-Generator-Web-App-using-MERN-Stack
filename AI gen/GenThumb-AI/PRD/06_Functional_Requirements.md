# 6. Functional Requirements

FR1: Users must be able to register an account.
FR2: Users must be able to log in.
FR3: Users must generate thumbnails using prompts.
FR4: System must store generated thumbnails.
FR5: Users must download thumbnails.
FR6: Users must edit thumbnails.
FR7: System must predict CTR for thumbnails.
FR8: Users must be able to view thumbnail history.
